Derivative Applications and Integrals

Derivative Properties Summary

Def: the tangent to \(f(x)\) at \(x=a\) is equal to $$\lim_{x\to a} \frac{f(x)-f(a)}{x-a} = \lim_{h\to 0} \frac{f(a+h)-f(a)}{h} = f'(a)$$

Def: \(f(x)\) is diffrentiable at \(x=a\) if \(f'(a)\) exists

Properties~

$$diffrentiability \implies continuity$$

$$\frac{d}{dx}(f(x)+g(x))=f'(x)+g'(x)$$

$$\frac{d}{dx}(kf(x))=kf'(x)$$

$$\frac{d}{dx}(c)=0$$

$$\frac{d}{dx}(x^n)=nx^{n-1}$$

$$\frac{d}{dx}(\sin{x})=\cos{x}$$

$$\frac{d}{dx}(\cos{x})=-\sin{x}$$

$$\frac{d}{dx}\left(\frac{1}{f(x)}\right)=-\frac{f'(x)}{(f(x))^2}$$

$$\frac{d}{dx}(f(x)g(x))=f'(x)g(x)+g'(x)f(x)$$

$$\frac{d}{dx}(f(g(x)))=f'(g(x))g'(x)$$

Proof Generalizations~ if you need to prove any of the derivative properties using limits,

1) convert definition to any of the limit definitions listed above

2) multiply to cancel out denominators or apply synthetic division

3) factor and cancel out numerator and denominator, or expand and cancel out h (depending on which definition you use)

Theorem: if \(f\) is continuous on \([a,b]\) and diffrentiable on \((a,b)\) and \(f(a)=f(b)\), then \(\exists \ c \in (a,b) \ s.t. f'(c)=0\)

Proof: Case 1 $$f(a)=f(b)=max\ of\ f\ on\ [a,b]$$ $$\exists\ min \in [a,b] = c \implies f'(c)=0 $$ Case 2 $$f(a)=f(b)=min$$ $$\exists\ max \in [a,b] = c \implies f'(c)=0 $$ Case 3 $$f(a)=f(b)=min\land max$$ $$\forall c \in [a,b], f'(c)=0 $$

Theorem: if \(f\) is continuous on \([a,b]\) and diffrentiable on \((a,b)\), then \(\exists \ c \in (a,b) \ s.t. f'(c)=\frac{f(b)-f(a)}{b-a}\)

Proof: $$Let\ g(x)= f(x)-f(a)-\frac{f(b)-f(a)}{b-a}(x-a)$$ $$\implies g(a)=0\land g(b)=0$$ $$\implies \exists a < c < b\ s.t.\ g'(c)=0 $$ $$g'(x)=f'(x)-\frac{f(b)-f(a)}{b-a}$$ $$\implies g'(c)=f'(c)-\frac{f(b)-f(a)}{b-a}$$ $$\implies 0=f'(c)-\frac{f(b)-f(a)}{b-a} $$ $$\implies f'(c)=\frac{f(b)-f(a)}{b-a}$$

Convexity

Def: A function \(f(x)\) is convex on \((a,b)\) if given \(a \leq r < s \leq b\), the secant line connecting \((r,f(r))\) to \((s,f(s))\) lies over the graph

Def: A function \(f(x)\) is concave on \((a,b)\) if given \(a \leq r < s \leq b\), the secant line connecting \((r,f(r))\) to \((s,f(s))\) lies under the graph (think mancave)

To derive symbolic definitions, draw a generic parabola and points a,b,\((c,f(c))\) inbetween the two points \((r,f(r))\) and \((s,f(s))\), then show that a generic point on the secant line \((c,?)\) is greater than the other point \((c,f(c))\). To prove any theorems concerning convexity, convert the convex definition into alternative form, and use the assumption and some generic points that utilize the assumption to eventually show the alternative form of the convex definition.

Derivative Tests

Def: First Derivative Test states that \(\forall a < c < b\ , f'(a) < 0 \land f'(b) > 0 \implies f(c)\) is a local min and \(\forall a < c < b\ , f'(a) > 0 \land f'(b) < 0 \implies f(c)\) is a local max

Def: Second Derivative Test states that \(f''(c) > 0 \implies f(c)\) is a local min and \(f''(c) < 0 \implies f(c)\) is a local max

Proof~

\(f''(c) > 0 \implies f'\) is increasing in a neighborhood of c

\(\implies f'\) goes from negative to positive

\(\implies f(c)\) is a local min by the First Derivative Test

Related Rates Problems

The strategy to solving problems is

0) Draw,

1) Given,

2) Want,

3) Relate (pythagorean thoerem,similar triangles,area / volume formulas,law of cosines ~ \(z^2=x^2+y^2-2xy\cos \theta \)),

4) Diffrentiate,

5) Solve (plug in),

6) State Answer (express written answer with correct units)

Optimization Problems

The strategy to solving problems is

0) Draw,

1) Given,

2) Want (minimize or maximize some expression),

3) Relate,

4) Diffrentiate (to find critical points),

5) Check (plug in endpoints, critical points, and find what you want, apply FDT or SDT on answer and confirm it is a min or max),

6) State answer (written answer with units)

Inverse Function Applications

Def: a function \(f(x)\) is monotonic on an interval if it is either increasing or decreasing on that interval

Theorem: If \(f(x)\) is increasing then so is \(f^{-1}(x)\)

Proof: $$(f(x)\ is\ increasing) \implies (x < y \implies f(x) < f(y)) $$ $$f(x)\geq f(y) \implies x \geq y $$ $$Let\ (f(x)=\tilde{x})\land (f(y)=\tilde{y}) $$ $$\implies (x=f^{-1}(\tilde{x})))\land (y=f^{-1}(\tilde{y}))$$ $$\tilde{x} \geq \tilde{y} \implies f^{-1}(\tilde{x}) \geq f^{-1}(\tilde{y})$$ $$\tilde{x} > \tilde{y} \implies f^{-1}(\tilde{x}) > f^{-1}(\tilde{y})$$ $$f^{-1}(x)\ is\ increasing$$

Theorem: If \(f(x)\) is continuous and increasing on \([a,b]\) then \(f^{-1}(x)\) is continuous on \([f(a),f(b)]\)
