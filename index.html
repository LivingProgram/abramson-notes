Derivative Applications and Integrals

Derivative Properties Summary

Def: the tangent to \(f(x)\) at \(x=a\) is equal to $$\lim_{x\to a} \frac{f(x)-f(a)}{x-a} = \lim_{h\to 0} \frac{f(a+h)-f(a)}{h} = f'(a)$$

Def: \(f(x)\) is diffrentiable at \(x=a\) if \(f'(a)\) exists

Properties~

$$diffrentiability \implies continuity$$

$$\frac{d}{dx}(f(x)+g(x))=f'(x)+g'(x)$$

$$\frac{d}{dx}(kf(x))=kf'(x)$$

$$\frac{d}{dx}(c)=0$$

$$\frac{d}{dx}(x^n)=nx^{n-1}$$

$$\frac{d}{dx}(\sin{x})=\cos{x}$$

$$\frac{d}{dx}(\cos{x})=-\sin{x}$$

$$\frac{d}{dx}\left(\frac{1}{f(x)}\right)=-\frac{f'(x)}{(f(x))^2}$$

$$\frac{d}{dx}(f(x)g(x))=f'(x)g(x)+g'(x)f(x)$$

$$\frac{d}{dx}(f(g(x)))=f'(g(x))g'(x)$$

Proof Generalizations~ if you need to prove any of the derivative properties using limits,

1) convert definition to any of the limit definitions listed above

2) multiply to cancel out denominators or apply synthetic division

3) factor and cancel out numerator and denominator, or expand and cancel out h (depending on which definition you use)

Inverse Function Applications

Def: a function \(f(x)\) is monotonic on an interval if it is either increasing or decreasing on that interval

Theorem: If \(f(x)\) is increasing then so is \(f^{-1}(x)\)

Proof: $$(f(x)\ is\ increasing) \implies (x < y \implies f(x) < f(y)) $$ $$f(x)\geq f(y) \implies x \geq y $$ $$Let\ (f(x)=\tilde{x})\land (f(y)=\tilde{y}) $$ $$\implies (x=f^{-1}(\tilde{x})))\land (y=f^{-1}(\tilde{y}))$$ $$\tilde{x} \geq \tilde{y} \implies f^{-1}(\tilde{x}) \geq f^{-1}(\tilde{y})$$ $$\tilde{x} > \tilde{y} \implies f^{-1}(\tilde{x}) > f^{-1}(\tilde{y})$$ $$f^{-1}(x)\ is\ increasing$$

Theorem: If \(f(x)\) is continuous and increasing on \([a,b]\) then \(f^{-1}(x)\) is continuous on \([f(a),f(b)]\)
